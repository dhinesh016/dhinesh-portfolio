# Dhinesh Portfolio (Full Stack)

This is my **full-stack portfolio project** built using React and Spring Boot.
It includes a contact form that stores user data in a database and sends email notifications.

---

## 🚀 Technologies Used

### Frontend

* React (Vite)
* Axios
* CSS

### Backend

* Java
* Spring Boot
* Spring Data JPA
* Maven

### Database

* PostgreSQL (Cloud)

### Deployment

* Render

---

## 📂 Project Structure

```
dhinesh-portfolio
├── frontend   # React application
└── backend    # Spring Boot application
```

---

## ⚙️ Features

* Responsive portfolio UI
* Contact form with API integration
* Stores messages in database
* Sends email notifications
* Full-stack deployment

---

## 🔗 API Endpoint

POST /contact

Example Request:

```
{
  "username": "Dhinesh",
  "email": "dhinesh@gmail.com",
  "mobile": "9876543210",
  "message": "Hello"
}
```

---

## 🛠️ How to Run Locally

### Frontend

```
cd frontend
npm install
npm run dev
```

### Backend

```
cd backend
./mvnw spring-boot:run
```

## 🌐 Deployment
